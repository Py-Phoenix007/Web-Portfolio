Project 1: Python Pair Finder

This project is a Python program I created to find pairs of numbers in a list that sum up to a given target, with no duplicate pairs. Itâ€™s a simple yet efficient way to practice logical thinking and Python programming.

Code:


def find_pairs(lst, target):
    lst = list(set(lst))
    output = []
    for i in range(len(lst)):
        complement = target - lst[i]
        if complement in lst[i+1:]:
            output.append((lst[i], complement))
    print(output)
            
lst = [1, 2, 3, 4, 3, 2, 1]
target = 5
find_pairs(lst, target)

Output:

This code outputs the pairs of numbers that add up to the target value without any duplicates.
